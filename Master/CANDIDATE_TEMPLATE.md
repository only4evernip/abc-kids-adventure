# CANDIDATE_TEMPLATE.md - Master

## 目标

定义 Master 的候选池输出模板。

候选池不是“把今天涨得好的票全列一遍”，而是把真正值得跟踪、试错、回避的对象分层输出。

核心要求：
- 先分层
- 再排序
- 再写动作
- 不允许把杂毛和核心混在一起

---

# 一、候选池总原则

## 候选池必须回答 4 个问题
1. 今天最值得看的票是谁？
2. 今天哪些票可以小仓试错？
3. 今天哪些票最危险、要回避？
4. 今天哪些票虽然强，但不属于高质量机会？

## 候选池不是涨幅榜
候选池的排序优先级必须是：
1. 是否顺环境
2. 是否顺主线
3. 是否是核心角色
4. 当前是不是好节点
5. 风险收益比是否合理

而不是：
- 谁涨得最多
- 谁最热闹
- 谁看起来最刺激

---

# 二、候选池分层结构

建议固定分成 4 组：

1. **Top 3 重点跟踪**
2. **Top 3 小仓试错**
3. **观察名单**
4. **高风险回避名单**

如果需要更完整，可加：
5. **Top 10 全量候选排序**

---

# 三、Top 3 重点跟踪模板

## 定义
这一组必须满足：
- 顺环境
- 顺主线
- 角色清晰
- 节点合理
- 不是情绪高潮末端

## 模板
```md
## Top 3 重点跟踪

### 1. [股票名称 / 代码]
- **角色定位：** 总龙头 / 板块龙头 / 情绪龙头 / 趋势中军 / 核心补涨
- **所属主线：** [板块名称]
- **当前阶段：** [主升 / 分歧 / 回踩确认 / 强转强]
- **入选理由：** [一句话写清楚为什么入选]
- **当前信号：** [分歧低吸 / 回踩确认 / 趋势跟随 / 打板确认]
- **建议动作：** 重点跟踪 / 可择机建仓 / 可趋势持有
- **风险提示：** [一句话]
- **失效条件：** [一句话]
```

## 排序规则
重点跟踪内部排序优先级：
1. 龙头 / 中军优先于补涨
2. 顺主线优先于边缘分支
3. 分歧良性优先于高潮一致
4. 结构健康优先于纯情绪脉冲

---

# 四、Top 3 小仓试错模板

## 定义
这一组适合：
- 新方向试错
- 主线边缘但尚未证伪
- 情绪修复早期
- 低位启动但仍需确认

## 模板
```md
## Top 3 小仓试错

### 1. [股票名称 / 代码]
- **角色定位：** 核心补涨 / 启动先锋 / 新方向试错标的
- **所属方向：** [板块 / 题材]
- **试错逻辑：** [一句话]
- **当前信号：** [试错买点 / 低位启动 / 修复确认]
- **建议动作：** 小仓试错
- **主要风险：** [一句话]
- **失效条件：** [一句话]
```

## 排序规则
小仓试错内部排序优先级：
1. 更顺当前环境的优先
2. 更接近主线的优先
3. 低位清晰结构优先于高位博弈
4. 有验证路径的优先于纯想象力

---

# 五、观察名单模板

## 定义
观察名单不是推荐买入，而是：
- 还差一个确认信号
- 位置不差，但时机没到
- 板块不错，但个股节点一般
- 个股强，但环境或主线仍需观察

## 模板
```md
## 观察名单

### 1. [股票名称 / 代码]
- **角色定位：** [角色]
- **所属方向：** [板块]
- **当前状态：** [观察原因]
- **需要确认：** [确认条件]
- **若确认成立：** [可能升级为重点跟踪 / 小仓试错]
- **若确认失败：** [降级为回避 / 继续观察]
```

## 观察名单常见场景
- 主线内中军走势还没完全转强
- 龙头刚经历大分歧，需观察次日承接
- 新热点第一天太一致，不适合直接追
- 个股位置偏高，需要等回踩确认

---

# 六、高风险回避名单模板

## 定义
这一组非常重要。不是陪衬，而是 Master 的核心价值之一。

必须重点列出：
- 退潮反抽票
- 高位一致末端票
- 龙头断板后的后排跟风
- 杂毛伪核心
- 板块不强但个股硬拉的诱多票

## 模板
```md
## 高风险回避名单

### 1. [股票名称 / 代码]
- **角色定位：** 跟风 / 杂毛 / 退潮反抽 / 高位博弈
- **风险类型：** [退潮反抽 / 龙头见顶后补涨 / 假突破 / 缩量加速末端]
- **回避理由：** [一句话写清楚]
- **建议动作：** 回避
```

## 回避名单排序规则
优先把最容易坑人、最容易诱多的票排前面。

# 八、排序总规则

Master 做候选池排序时，默认优先级如下：

## 一级优先级
- 是否顺环境
- 是否顺主线
- 是否是核心角色

## 二级优先级
- 当前是不是好节点
- 结构是否健康
- 风险收益比是否合理

## 三级优先级
- 市场辨识度
- 成交额承接能力
- 是否容易被后续验证

## 降级规则
满足以下任一项，候选池必须降级：
- 环境差
- 板块退潮
- 个股角色不清
- 高位爆量
- 失效条件模糊

---

# 九、候选池输出禁忌

## 1. 禁止杂毛混入重点跟踪
只要角色不清或明显跟风，不能进重点池。

## 2. 禁止把“今天涨得猛”当作唯一入选条件
涨得猛不等于能做。

## 3. 禁止不给失效条件
只给看多理由，不给失效条件，等于没说。

## 4. 禁止不给回避名单
没有回避名单的候选池，说明系统还不够成熟。

---

# 十、理想效果

一份好的候选池输出，应该让人一眼看明白：

1. 今天最该盯哪 3 只
2. 今天可以试错哪 3 只
3. 今天最容易踩坑的是哪几只
4. 今天哪些票虽然热闹，但不值得做

如果做不到这几点，这份候选池就不合格。
