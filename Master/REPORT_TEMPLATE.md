# REPORT_TEMPLATE.md - Master

## 目标

定义 Master 的标准日报输出模板。

这个模板的核心不是“写得像财经媒体”，而是：
- 先给结论
- 再给结构
- 最后给动作
- 让人一眼知道今天该不该做、该做什么、不要做什么

---

# 一、日报总原则

## 输出要求
- 先结论，后原因
- 少废话，不讲空泛大道理
- 用固定结构，避免每天风格漂移
- 明确写出风险，不做暧昧表达
- 输出必须服务于动作，而不是服务于情绪

## Master 的日报必须回答 4 个问题
1. 今天环境能不能做？
2. 今天主线是谁？
3. 今天哪些票值得看？
4. 今天哪些坑必须躲？

# 二、标准日报结构

## Part 1：今日总览（必须放最前面）

### 模板
```md
# Master 日报｜YYYY-MM-DD

## 今日一句话结论
[一句话写清楚今天的市场状态和行动原则]

## 今日环境结论
- **环境标签：** 进攻 / 试错 / 混沌 / 防守
- **情绪周期：** 冰点 / 修复 / 主升 / 高潮 / 强分歧 / 退潮 / 混沌
- **综合判断：** [一句话]
- **今日适合：** [趋势低吸 / 核心跟随 / 小仓试错 / 防守等待]
- **今日严禁：** [追高后排 / 接退潮反抽 / 杂毛套利 / 无脑打板]
```

### 要求
这一部分必须让人不看后文，也知道今天大概该怎么活。

## Part 2：市场环境（Level 1）

### 模板
```md
## Level 1｜市场环境
- **指数结构：** [强 / 中 / 弱]
- **成交额状态：** [放量 / 缩量 / 存量博弈]
- **赚钱效应：** [强 / 一般 / 弱]
- **亏钱效应：** [低 / 可控 / 扩散]
- **市场评分：** XX/100

### 关键观察
- [观察点 1]
- [观察点 2]
- [观察点 3]

### 环境判断
[用 2-4 句解释为什么是进攻 / 混沌 / 防守]
```

### 要求
必须写清：
- 为什么现在能做 / 不能做
- 风险来自哪里
- 当前到底是“系统性机会”还是“结构性机会”

# 五、日报写作禁忌

Master 的日报禁止出现这些问题：

## 1. 禁止空话
例如：
- “市场或将迎来新的机会”
- “后续值得持续关注”
- “建议投资者理性看待市场波动”

这些都没用。

## 2. 禁止模糊表达
例如：
- “可以适当关注”
- “不排除后续继续走强”
- “短期存在一定机会”

必须明确：
- 观察 / 试错 / 建仓 / 回避 / 减仓 / 退出

## 3. 禁止先讲个股，后讲环境
顺序不能乱。

## 4. 禁止把跟风写得像核心
角色判断必须清楚。

# 六、Master 日报的理想效果

一份好的 Master 日报，应该让人 30 秒内就知道：

1. 今天能不能做
2. 今天该做哪条线
3. 今天该看哪几只
4. 今天最容易踩什么坑

如果做不到这 4 点，这份日报就不合格。
